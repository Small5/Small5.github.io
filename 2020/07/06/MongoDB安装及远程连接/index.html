ubuntu18.04安装MongoDB并使用compass远程连接

Published 2020-07-06

今天在实验室服务器上安装测试了redis和mongodb并测试，现介绍一下mongodb的过程。

安装配置

1、安装MongoDB
apt-get install mongodb
2、安装好后执行mongo进入数据库命令行
mongo
3、查看数据库，和MySQL类似
show dbs

4、新建数据库，直接use，不存在则会新建
use myNewDatabase
5、添加用户，其中role字段有几种参数，可以参考官方文档具体了解

db.createUser(
… {
… user: “Small5”,
… pwd:”123456”,
… .roles: [{role: “dbAdmin”, db: “myNewDatabase”}]
… }
… )
6、插入一条记录
db.myCollection.insertOne({X : 1})
7、编辑配置文件，注释绑定本地ip,开启远程连接。
sudo vi /etc/mongodb.conf
#bind ip = 127.0.0.1
