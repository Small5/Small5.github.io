Redis安装及简要介绍

By Patrick Young

Published 2020-07-06

在线安装

直接输入命令 sudo apt-get install redis-server

安装完成后，Redis服务器会自动启动。
使用ps -aux|grep redis命令可以看到服务器系统进程默认端口6379

使用netstat -nlt|grep 6379命令可以看到redis服务器状态

使用sudo /etc/init.d/redis-server status命令可以看到Redis服务器状态
